임대인 법인 건전성 분석기 — 완전 가이드

앱 바로가기

이 앱은 무엇인가요?

법인체크는 임대인이 법인(회사)일 때 그 법인의 재무제표, 부채비율, 등기 변동 이력을 분석하여 전세사기 법인 패턴에 해당하는지 자동 판별하고, 계약 가능 여부를 AI가 종합 진단해주는 도구입니다.

왜 법인 임대인을 확인해야 하나요?

  • 전세사기의 70% 이상이 법인(갭투자 회사) 소유 부동산에서 발생
  • 자본금 1억 미만으로 수십~수백 채 보유하는 깡통 법인이 핵심 위험
  • 대표이사를 자주 바꿔 책임 회피하는 패턴이 전형적
  • 개인 임대인과 달리 파산 시 보증금 회수 극히 어려움

1. 화면 구성

A
왼쪽 패널 — 법인 검색, 위험/주의/안전 필터, 법인 리스트 카드
B
오른쪽 탭 4개
  • 건전성 대시보드 — 종합 점수, 핵심 지표, 경고, 등기 변동 이력
  • DART 재무 공시 — 재무상태표, 3년 손익, 부채비율/유동비율 게이지
  • 네이버 뉴스/블로그 — 법인명으로 실시간 뉴스 검색
  • AI 종합 진단 — 건전성 등급 + 전세사기 패턴 매칭 + 필수 특약

2. 주요 기능 사용법

1 법인 검색 & 선택

  1. 검색창에 법인명 또는 사업자등록번호 입력
  2. 필터 버튼으로 위험/주의/안전 법인만 표시 가능
  3. 법인 카드 클릭 → 오른쪽에 상세 분석 대시보드 표시
  4. 카드 왼쪽 색상 바: 🟢 안전 / 🟡 주의 / 🔴 위험

2 건전성 대시보드 읽기

종합 점수 — 100점 만점 게이지 바

  • 80점+ = 안심 | 50~79점 = 주의 | 50점 미만 = 위험

핵심 지표 4가지

  • 자본금 — 1억 이하이면서 부동산 다수 보유 시 위험
  • 부채비율 — 200% 이하 안전, 300%+ 경고, 500%+ 위험
  • 유동비율 — 100% 이상 안전, 미만이면 단기 지급 능력 부족
  • 보유 부동산 — 자본금 대비 과다 보유 시 갭투자 의심

위험/주의 사항 — 빨간/노란 박스로 구체적 경고 표시

등기 변동 이력 — 대표이사 변경, 본점 이전, 자본금 변경 등 타임라인

3 DART 재무 공시 탭

  1. "DART 재무 공시" 탭 클릭
  2. 재무상태표: 총자산 / 총부채 / 자본총계 한눈에 확인
  3. 3년 추이: 매출액과 당기순이익 변화 (적자 시 빨간색 표시)
  4. 부채비율·유동비율 게이지 바 — 안전기준 대비 시각 비교
핵심: 부채비율 200% 초과 + 3년 연속 적자 = 전세 계약 비추천

4 AI 종합 진단 (핵심 기능)

  1. 법인 선택 후 대시보드 하단 "이 법인 AI 종합 건전성 진단" 클릭
  2. 또는 "AI 종합 진단" 탭에서 법인 정보 직접 입력
  3. 분석 결과:
    • 건전성 등급 S ~ F (S=최우수, F=위험)
    • 전세 계약 추천 — 안전/주의 후 계약/비추천/강력 비추천
    • 전세사기 패턴 체크표 — 5가지 패턴 해당 여부 표로 표시
    • 보증보험 가입 가능성 — 가능/조건부/불가 예상
    • 필수 특약 조항 3개 — 그대로 계약서에 복사 가능

5 네이버 뉴스 검색

  1. 법인 선택 시 자동으로 법인명으로 네이버 뉴스 검색
  2. 위험 키워드(사기, 부도, 압류, 횡령, 파산 등) 감지 시 빨간 주의 배지
  3. 뉴스 카드 클릭 → 원문 페이지로 이동

3. 전세사기 법인 5대 패턴

패턴설명위험도
대표이사 잦은 변경2년 내 2회 이상 → "바지사장" 세워 책임 회피매우 높음
자본잠식부채가 자본 초과 → 보증금 반환 재원 없음매우 높음
대부업 사업목적사업목적에 "대부업" 포함 → 고금리 자금 조달매우 높음
소자본 다수 부동산자본금 1억으로 100채+ → 전형적 갭투자 법인매우 높음
잦은 본점 이전1년 내 2회+ → 채권자 추적 회피높음
