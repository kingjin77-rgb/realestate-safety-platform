경매알리미는 법원 경매 물건을 지도 히트맵으로 한눈에 파악하고, 낙찰가율 시뮬레이터로 투자 수익을 예측하며, AI가 권리분석·명도 난이도·투자 등급을 자동 산출해주는 경매 투자 보조 도구입니다.

1. 화면 구성

A
왼쪽 패널 — 검색, 물건 필터, 통계 카드, 낙찰가 시뮬레이터, 물건 리스트, Datalab 트렌드, 뉴스 피드
B
오른쪽 탭
  • 경매 히트맵 — 물건 위치 + 위험도 마커
  • AI 투자 분석기 — 물건 정보 입력 → 투자 등급 산출
  • 물건 상세 정보 — 감정가/근저당/임차인 등 상세

2. 주요 기능 사용법

1 물건 검색 & 필터

  1. 검색창에 지역명, 아파트명, 사건번호 입력 후 Enter
  2. 빠른 필터 버튼: 아파트 / 빌라 / 오피스텔 / 토지 / 전체
  3. 정렬 옵션: 매각기일순 / 감정가순 / 낙찰가율순
  4. 물건 카드 좌측 색상 바: 🟢 양호 / 🟡 주의 / 🔴 위험

2 낙찰가율 예측 시뮬레이터

  1. 감정가 슬라이더 — 5천만원 ~ 20억 범위 조절
  2. 유찰 횟수 슬라이더 — 유찰 1회당 최저가 20% 하락
    • 0회: 100% | 1회: 80% | 2회: 64% | 3회: 51.2%
  3. 경쟁 입찰자 수 — 많을수록 예상 낙찰가 상승
  4. 하단에 최저입찰가 / 예상 낙찰가 / 할인율 자동 계산

3 물건 상세 보기

  1. 왼쪽 물건 리스트에서 카드 클릭 → 상세 정보 탭 자동 전환
  2. 감정가, 최저입찰가, 유찰 횟수, 매각기일, 임차인 수, 근저당 합계 표시
  3. 하단 "이 물건 AI 투자 분석하기" 버튼 → AI 분석 탭으로 자동 이동

4 AI 투자 분석 (핵심 기능)

  1. "AI 투자 분석기" 탭 클릭
  2. 샘플 3개 중 선택하거나, 물건 상세에서 자동 입력되거나, 직접 입력
  3. "AI 경매 투자 분석 시작" 클릭
  4. 결과 보고서:
    • 투자 등급 S/A/B/C/D — 위험/수익 종합 판단
    • 위험 요소 — 깡통, 미배당 임차인, 유치권 등
    • 예상 낙찰가 — 유찰 횟수 기반 산출
    • 명도 난이도 — 쉬움/보통/매우 어려움
    • 투자 체크리스트 — 입찰 전 확인 항목

5 경매 뉴스 & Datalab 트렌드

  1. "실시간 검색" 버튼 → 네이버에서 "법원 경매 낙찰", "경매 투자 후기" 실시간 검색
  2. 뉴스 카드 클릭 → 원문 페이지로 이동
  3. Datalab: "법원경매", "경매 낙찰가율", "캠코 공매" 90일 트렌드
