이 앱은 무엇인가요?

우동전은 전세 계약 전 해당 지역의 법원 등기 통계, 네이버 실시간 커뮤니티 반응, 검색 트렌드를 한눈에 확인하고, 계약서 특약사항의 위험도를 AI가 즉시 분석해주는 전세사기 예방 도구입니다.

1. 화면 구성

A
왼쪽 패널 — 지역 검색, 안전점수, 시뮬레이터, 등기 통계, 네이버 트렌드, 실시간 피드
B
오른쪽 상단 탭 — 3개 탭 전환 가능
  • 등기 안심 지도 — 지역별 안전도 마커가 표시된 지도
  • AI 등기·특약 분석기 — 계약서 텍스트 입력 → 즉시 위험 분석
  • 등기관장 API 콘솔 — 법원 API 직접 호출 테스트

2. 주요 기능 사용법

1 지역 검색

  1. 왼쪽 상단 검색창에 아파트 이름 또는 동 이름 입력 (예: 마포 상암, 목동, 화곡동)
  2. Enter 또는 검색 버튼 클릭
  3. 해당 지역의 등기 통계 + 안전점수 + 네이버 피드가 자동 로드됨
  4. 지도의 마커를 클릭해도 해당 지역으로 전환됨

2 안전점수 시뮬레이터

  1. 임차권등기명령 신청 수 슬라이더 — 건수 올리면 점수 급락 (건당 -15점)
  2. 경매 매각 처분 수 슬라이더 — 건수 올리면 점수 하락 (건당 -10점)
  3. 제2금융권 비율 슬라이더 — 20% 초과 시 감점 시작
  4. 슬라이더 조절 즉시 → 점수 + 지도 마커 색상이 실시간 변경됨
점수 기준: 80점+ = 🟢 안심 | 50~79점 = 🟡 주의 | 50점 미만 = 🔴 위험

3 네이버 실시간 피드 & Datalab 트렌드

  1. 네이버 API 키가 내장되어 있어 바로 동작
  2. 실시간 검색 버튼 → 해당 지역명 + "전세" 키워드로 블로그/뉴스/카페 검색
  3. 위험 키워드(사기, 경매, 압류 등) 감지 시 빨간 배지 자동 표시
  4. Datalab 트렌드: "전세사기", "역전세", "전세보증보험" 등 90일 검색량 표시

4 AI 계약서 특약 분석 (핵심 기능)

  1. 상단 "AI 등기·특약 분석기" 탭 클릭
  2. 샘플 버튼 3개 중 하나 클릭하거나, 실제 계약서 특약사항 직접 입력
  3. "AI 즉시 권리분석 시작하기" 버튼 클릭
  4. 결과 보고서 확인:
    • 🔴 위험 사항 — 즉시 수정 필요 조항
    • 🟡 주의 사항 — 확인 필요 조항
    • 🟢 안심 조항 — 잘 구성된 보호 조항
    • 📋 권장 보완 특약 — 그대로 계약서에 복사 가능
팁: AI가 제공하는 "권장 보완 특약"을 그대로 복사해서 실제 계약서에 넣으세요!

5 법원 등기 API 콘솔

  1. 상단 "등기관장 API 실시간 콘솔" 탭 클릭
  2. API 종류 선택 (확정일자/소유권이전/근저당설정/근저당말소)
  3. 행정구역 코드 입력 (예: 11440 = 마포구)
  4. "API 호출 샌드백 전송" 버튼 → 터미널에 실시간 응답 표시

3. API 키 설정 (선택)

네이버 API 키는 기본 내장되어 있어 별도 설정 없이 사용 가능합니다.

직접 발급받은 키로 변경하려면:

  1. 우측 상단 "API 키 현황" 버튼 클릭
  2. 네이버 Client ID / Secret 입력
  3. 법원 IROS API 키 입력 (선택)
  4. "전체 키 저장" 클릭 → 초록색 "저장됨 ✓" 확인

저장된 키는 브라우저 localStorage에 보관 — 재방문 시 자동 복원
